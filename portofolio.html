Wellness Health App

A comprehensive mobile and desktop application for treatment reservations, offering users an easy way to book and manage their wellness treatments. Developed with Flutter and a .NET Core Web API (RESTful) for a seamless cross-platform experience. The app uses SignalR for real-time communication between the mobile and desktop applications. Notifications about bookings and updates are sent via RabbitMQ, and an external mail service is used to send confirmation and reminder emails to users.

Technologies used: Flutter (Dart), .NET Core Web API (RESTful), RabbitMQ, Microsoft SQL

Key Features: Personalized health tracking, user profile management, interactive data visualization, and secure payment integration.

Challenges: Ensuring smooth synchronization across mobile (Android), and Desktop platforms. Implementing a secure payment system via Stripe and PayPal while managing high traffic loads during peak usage.
